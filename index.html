FAQ

What people usually want to know first.

What is Intro Deck?

Intro Deck is a private professional directory and contact workflow that runs inside Telegram.

Is this public?

No. Directory visibility and contact access are controlled inside the product and do not default to public contact exposure.

What does LinkedIn connect import?

The basic identity layer, such as name and other supported identity fields from the current sign-in contract. Public profile fields are still managed in Telegram.

Do you scrape LinkedIn?

No. Intro Deck uses LinkedIn sign-in as the identity layer. It is not a scraping tool and not a LinkedIn clone.

How do intros and direct contact work?

Depending on the profile settings, you can request an intro, request direct contact, or use the private DM relay path inside Telegram.

Is direct contact always visible?

No. Contact access can stay hidden, approval-based, or routed through private DM depending on the profile contract.

Why does it run in Telegram?

Because discovery and follow-up are more useful when the workflow stays close to the actual conversation layer.

What happens after I connect LinkedIn?

You return to Telegram, finish your card, decide how visible it should be, and start using the directory workflow.
