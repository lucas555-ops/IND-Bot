# 05_ARTIFACT_PROTOCOL

Every real STEP should end with a file set, not only prose.

## Required artifacts

- FULL ZIP
- Hotfix ZIP
- PATCH
- changed files list
- QA checklist

## What each means

### FULL ZIP
Full project snapshot after the STEP.

### Hotfix ZIP
Only files changed by the current STEP.

### PATCH
Concrete diff for the current STEP.

### changed files list
Short exact list of changed files.

### QA checklist
What was checked, what was not, what still needs manual/live verification.

## Forbidden endings

- “done conceptually”
- “roughly like this”
- “assemble it yourself later”
- “no patch, but logic is described”
