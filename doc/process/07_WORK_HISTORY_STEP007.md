# WORK_HISTORY_STEP007

## What changed

STEP007 added the first public browse layer for the directory.

Added:
- home entrypoint to public browse
- list surface for `listed + active` profiles
- public profile card surface
- page-preserving back navigation
- empty/degraded directory states
- directory smoke coverage

## Why this STEP was needed

STEP006 let users become directory-ready, but there was still no actual place where listed profiles could be browsed. STEP007 closes that gap without prematurely adding search, filters, premium, or contact actions.

## What was intentionally not done

- no search and filters yet
- no contact or intro flows
- no premium
- no moderation queue
- no ranking experimentation
