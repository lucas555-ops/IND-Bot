# STEP049C — OG / SOCIAL / METADATA UPLIFT

## Goal

Add a production-grade share-preview and metadata layer to the Intro Deck landing.

## Scope

- branded OG image
- full `og:` and `twitter:` metadata on the public landing
- canonical/title/description polish
- favicon + apple-touch icon consistency
- `robots.txt` and `sitemap.xml`
- narrow legal-page metadata alignment

## Out of scope

- further landing section redesign
- new landing copy architecture
- animated social assets
- multi-language metadata variants

## Product truth

- the landing remains a clean one-page product entry page
- OG/social should improve Telegram and social-link previews without changing the core product contract
- metadata should stay sober, product-grade, and consistent with the LinkedIn-as-identity-layer truth

## Acceptance

- homepage includes full `og:` and `twitter:` meta tags
- homepage points to a branded OG image under `assets/social/`
- homepage/legal pages include canonical + favicon metadata
- `robots.txt` and `sitemap.xml` exist
- metadata remains consistent with the current product positioning
